# n8n Community Node: Hack AI

This is an n8n community node for [Hack AI](https://ai.hackclub.com), an OpenAI-compatible API by Hack Club.

## Features
- Connect to ai.hackclub.com for AI tasks
- Compatible with OpenAI-style endpoints
- Easy integration with n8n workflows

## Installation

Install via npm:

```bash
npm install n8n-nodes-hackai
```

## Usage

Add the Hack AI node to your workflow and configure your credentials for ai.hackclub.com.
